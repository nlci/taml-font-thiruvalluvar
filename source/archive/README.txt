NLCI Tamil fonts
================

This file describes the legacy font files included with the NLCI Tamil fonts.
This information should be distributed along with the NLCI Tamil fonts and any derivative works.

This Font Software is licensed under the SIL Open Font License, Version 1.1.

TrueType *.ttf files:
- Original legacy, hacked symbol encoded fonts that supplied glyphs and names for the new Unicode fonts.
- These legacy font files are intended only for development, not for use in displaying text in a program.
